# OneTab 备份
导出 [OneTab](http://www.one-tab.com/) 中收藏的网页。

意图：
* 预防因 OneTab 或浏览器错误，意外丢失收藏的网页。
* 减少浏览器内存占用。OneTab 有过多收藏占内存。
* 更便于整理长期积累的收藏列表。

## 功能

- 导入导出
  - [ ] OneTab 默认导出/导出格式
  - [ ] OneTab 内部在浏览器 Local Storage 中使用的格式
  - [x] 本应用使用的 SQLite 格式
- UI
  - [ ] 命令行
  - [ ] GUI
- 管理收藏列表
  - [ ] 合并
  - [ ] 整理
